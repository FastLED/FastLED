fl::asio::error_code Struct Reference

Detailed Description

Asio-compatible error code: numeric code + optional human-readable message.

Maps to boost::system::error_code in Asio.

Unlike fl::task::Error (string-only, heap allocation on every error), error_code is cheap to copy and supports quick boolean checks: if (ec) { /* error *‍/ }
