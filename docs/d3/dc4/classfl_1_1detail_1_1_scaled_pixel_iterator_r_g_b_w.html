fl::detail::ScaledPixelIteratorRGBW Class Reference

Detailed Description

Input iterator adapter for PixelIterator yielding 4-byte pixel data.

Similar to ScaledPixelIteratorRGB but yields 4-byte pixel arrays (RGBW).

Note
Yields fl::array<u8, 4> representing bytes in wire order (color order already applied)
