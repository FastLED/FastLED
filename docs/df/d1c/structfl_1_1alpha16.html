Detailed Description

Unsigned 16-bit alpha / brightness — UNORM16.

Represents values in [0.0, 1.0] inclusive.

Interpretation: raw / 65535 raw 0 = 0.0 raw 32768 = 32768/65535 ~ 0.500 raw 65535 = 65535/65535 = 1.0
