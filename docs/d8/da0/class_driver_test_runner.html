Detailed Description

Main test runner that orchestrates all driver tests.

Example usage:

CRGB leds[NUM_LEDS];
FastLED.addLeds<WS2812, DATA_PIN, GRB>(leds, NUM_LEDS);
DriverTestRunner runner(leds, NUM_LEDS);
runner.runAllTests();
runner.printSummary();
