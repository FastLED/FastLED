fl::NamedTimingConfig Struct Reference

Detailed Description

Chipset timing configuration with name for testing.

Carries timing + the byte-level encoder selector together because callers build runtime-named chipset variants (e.g., "UCS7604-800KHZ") and need both fields to construct a faithful ClocklessChipset downstream.
