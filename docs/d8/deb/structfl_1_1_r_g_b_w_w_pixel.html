fl::RGBWWPixel Struct Reference

Detailed Description

Simple 5-channel pixel struct: red, green, blue, warm-white, cool-white.

Note
This represents raw RGBWW pixel data (5 bytes per pixel).
Not to be confused with Rgbww which is a configuration struct.
