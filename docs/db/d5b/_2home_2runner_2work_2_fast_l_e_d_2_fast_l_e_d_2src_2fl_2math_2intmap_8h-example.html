Generic integer scaling between different integer sizes.

Generic integer scaling between different integer sizes.

Template Parameters
INT_FROMInput integer type (must be explicitly specified)
INT_TOOutput integer type (uint8_t, int8_t, uint16_t, int16_t, uint32_t, int32_t)
Parameters
xThe input value to scale
Returns
The scaled value preserving relative position in range

Requires explicit specification of BOTH input and output types to prevent implicit type conversions that could mask bugs. Uses bit replication for scaling up (8→16, 8→32, 16→32) and right-shift with rounding for scaling down (16→8, 32→16, 32→8).

Supports all combinations of 8-bit, 16-bit, and 32-bit signed/unsigned types. Compile-time error for unsupported type conversions.

uint16_t y = fl::int_scale<uint8_t, uint16_t>(102); // Explicit: 8→16
int32_t z = fl::int_scale<int8_t, int32_t>(64); // Explicit: 8→32
uint8_t w = fl::int_scale<uint32_t, uint8_t>(0x80000000); // Explicit: 32→8
uint8_t noop = fl::int_scale<uint8_t, uint8_t>(255); // Identity: no scaling
