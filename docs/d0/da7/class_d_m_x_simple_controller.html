Detailed Description

template<fl::u8 DATA_PIN, EOrder RGB_ORDER = RGB>
class DMXSimpleController< DATA_PIN, RGB_ORDER >

DMX512 based LED controller class, using the DmxSimple library.

Template Parameters
DATA_PINthe data pin for the output of the DMX bus
RGB_ORDERthe RGB ordering for these LEDs
