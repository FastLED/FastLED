fl::detail::ScaledPixelIteratorRGBWW Class Reference

Detailed Description

Input iterator adapter yielding 5-byte RGBWW pixels (issue #2558).

Mirrors ScaledPixelIteratorRGBW but produces 5 bytes per pixel (R, G, B, warm-W, cool-W in EOrder + EOrderWW wire order).
